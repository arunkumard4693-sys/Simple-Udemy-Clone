# Simple-Udemy-Clone

Welcome!
This is a simple Udemy clone website built using basic HTML and CSS.  
The project replicates the basic layout and design of the Udemy homepage.

## Features
- Homepage layout similar to Udemy  
- Course cards with title and price  
- Simple navigation bar  
- Built using only HTML and CSS  
- Beginner-friendly project

- CSS
